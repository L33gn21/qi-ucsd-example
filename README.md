# qi-ucsd-example

A small example project used for a QI @ UCSD demo. It contains a static landing page for a fictional taco shop.

## Contents

- [`best-taco-san-diego/`](best-taco-san-diego) — a static HTML/CSS landing page ("Best Taco San Diego")
  - `index.html` — page markup
  - `styles.css` — page styling

## Running locally

This is a static site with no build step or dependencies. To view it locally, open the HTML file directly in a browser:

```bash
open best-taco-san-diego/index.html
```

Or serve it with any static file server, e.g.:

```bash
npx serve best-taco-san-diego
```
